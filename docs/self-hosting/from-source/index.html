Build from Source

Prerequisites#

  • Google Cloud Project
  • BigQuery API Enabled
  • Cloud Storage Bucket
  • Service account credentials with access to all above
  • Mapbox Token

Steps#

  1. Checkout Dekart from GitHub; navigate to project directory;

  2. Install frontend dependencies

npm install
  1. Create and edit .env; see environment variables for details
cp .env.example .env
  1. Run Postgres DB locally
docker-compose  --env-file .env up
  1. Run Server; you will need to install godotenv or handle environment variable otherwise
godotenv -f .env go run ./src/server/main.go
  1. Run frontend
npm start
